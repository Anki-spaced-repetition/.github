# Anki Software — Spaced Repetition Flashcard System for Medical Students and Learners

## About Anki Software

Anki Software provides a complete spaced repetition flashcard system for medical students, language learners, exam candidates, and lifelong learners needing efficient long-term memorization. Anki Software schedules flashcard reviews at optimal intervals using active recall testing and memory algorithms. Anki Software creates cards with text, images, audio clips, video, and LaTeX mathematical notation. Anki Software syncs across devices (macOS, iOS, Android, Windows), downloads shared decks (GMAT, MCAT, USMLE, bar exam, languages), and adds community-developed add-ons. Anki Software offers close deletion, image occlusion, night mode, and statistics dashboard. Anki Software is trusted by medical students, law students, graduate exam candidates, polyglots, and professional certification seekers worldwide.

## Deep Dive — Memory Algorithm Performance Notes
SM-2 algorithm modification (Anki's native), ease factor adjustment, and when to reset card ease after many lapses
Review limit configuration (new cards per day, maximum reviews per day), learn ahead limit, and fuzz factor randomization for near-due cards
Card template customization (HTML, CSS), cloze deletion notation ((c1::text)), and image occlusion generation for labeled diagrams
Add-on compatibility (Heatmap, Review Retention, Image Occlusion Enhanced, Frozen Fields) and add-on conflict resolution
